About

This website has been developed as a minor project for college, aimed at streamlining the process of managing and accessing student marks. The platform is designed to serve two primary user roles: Students and Teachers.

Key Features and Functionalities

For Students :

  • Accessing Marks : Students can view their marks by entering their roll number securely.
  • Data Privacy : Each student can only view their own academic records.

For Teachers :

  • Registration and Login : Teachers can register and log in with secure credentials.
  • Subject Management : Teachers can add or remove subjects and select the ones they teach.
  • Batch-Wise Marks Allocation : Teachers can assign marks to students in batches for specific subjects.
  • Student Management : Add or remove students to keep the records up-to-date.
  • Role Management : Admin teachers can manage other teachers by assigning or revoking admin privileges.

..... and many more features are yet to be implemented.

This platform addresses common challenges in educational institutions by digitizing marks management. It eliminates manual errors, saves time, and ensures data privacy and security. The intuitive design and comprehensive features make it a reliable and scalable solution for academic purposes.

Technologies Used

  • Frontend : HTML, CSS, JavaScript, and Bootstrap for a responsive and visually appealing interface.
  • Backend : Django framework for efficient request handling and user authentication.
  • Database : PostgreSQL for secure and robust data storage.

It features a robust and user-friendly interface built with HTML, CSS, JavaScript, and Bootstrap for the frontend, while the backend is powered by Django and the database operations are handled using PostgreSQL.
